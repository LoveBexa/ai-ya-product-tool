# AIYA — Canva / presentation brand spec

Copy the block below into Canva AI when generating slides, social graphics, or pitch decks.

---

## Canva AI prompt (paste this)

Design a presentation for **AIYA**, an AI product planning tool. Follow this brand system exactly.

### Brand personality
Playful but professional. Light, creative, “clarity before code.” Pastel accents on soft lavender-grey backgrounds. Rounded, friendly UI — no sharp corporate blues.

### Fonts (Google Fonts)
| Role | Font | Weight | Style |
|------|------|--------|-------|
| **Headings & all body text** | **Poppins** | 600 (Semibold) for headings; 400–500 for body | Normal |
| **Accent word in headings** | **Playfair Display** | 500 | *Italic only* — use sparingly for one emphasised word (e.g. “*blueprint.*”) |
| **Logo wordmark** | Poppins | 700 (Bold) | ALL CAPS “AIYA”, letter-spacing +0.14em |

**Do not use:** Inter, Roboto, Arial, or heavy display fonts.

### Colour palette (hex)

**Core surfaces (app / light slides)**
- Background: `#E5E2F1` — soft lavender grey
- Card / white panels: `#FDFDFF`
- Light section alt: `#FAFAF9` or `#F4F4F7`
- Body text: `#171721`
- Muted / secondary text: `#62626F`
- Borders & dividers: `#D8D6DF`

**Dark hero (marketing homepage)**
- Dark background: `#0A0A0F`
- Hero body text: white at ~60% opacity (`#FFFFFF` @ 60%)
- Subtle caption text: white at ~40% opacity

**Primary actions (default buttons)**
- Button fill: `#1A1A21` — near-black charcoal
- Button text: `#F8F8FC` — off-white
- Shape: **fully rounded pill** (9999px / stadium shape)
- Height feel: ~40–48px; padding generous horizontally

**Accent CTA (secondary marketing buttons)**
- Button fill: `#76E1A7` — bright mint green
- Button text: `#082B1B` — deep forest green
- Same pill shape; font Semibold (600)

**Brand green (logo asterisk & highlights)**
- Logo on light bg: `#67CC95`
- Logo on dark bg: `#6CD79D`
- UI mint accent: `#76E1A7`
- Mint text on mint panels: `#082B1B`

**Secondary accents (use one per slide max)**
- Lilac: `#CFBCF4` (bg) / `#2D233E` (text on lilac)
- Yellow: `#F4E886` / `#42370D`
- Leaf green: `#C2EDAA` / `#213A1C`
- Pink: `#FFB6CB` / `#4B2532`

**Gradients (hero / emphasis only)**
- Purple → mint text gradient: `#C084FC` → `#A78BFA` → `#76E1A7`
- Mint feature section: `#E8FFC4` → `#76E1A7` (70%) → `#D4F7A8`
- Dark hero glow: soft purple radial at top (`rgba(120,80,255,0.35)`)

### Typography scale

| Element | Font | Size | Weight | Colour | Notes |
|---------|------|------|--------|--------|-------|
| **Hero H1** | Poppins | 48–60px | 600 | White (dark slide) or `#171721` (light) | Tight tracking (−0.02em). Short punchy lines. |
| **Section H2** | Poppins | 30–36px | 600 | `#171721` or `#082B1B` on mint sections | Centre or left align |
| **Card / step H3** | Poppins | 16–18px | 600 | `#171721` | |
| **Eyebrow / label** | Poppins | 11px | 600 | Mint `#76E1A7` or muted `#62626F` | ALL CAPS, letter-spacing +0.14em, pill badge |
| **Body / subheader** | Poppins | 14–16px | 400 | `#62626F` or white @ 60% on dark | Line-height ~1.6 (relaxed) |
| **Small caption** | Poppins | 12px | 400 | `#62626F` or white @ 40% | |
| **Footer** | Poppins | 12px | 400 | `#62626F` | |

### Buttons

| Type | Background | Text | Border | Radius | Font |
|------|------------|------|--------|--------|------|
| **Primary** | `#1A1A21` | `#F8F8FC` | none | Pill | Poppins 500, 14px |
| **Accent / CTA** | `#76E1A7` | `#082B1B` | none | Pill | Poppins 600, 14–16px |
| **Secondary** | `#F2F1F8` | `#23232E` | none | Pill | Poppins 500, 14px |
| **Outline** | `#FDFDFF` | `#171721` | 1px `#D8D6DF` | Pill | Poppins 500, 14px |
| **Ghost / nav link** | transparent | `#62626F` or white @ 70% | none | Pill | Poppins 400, 14px |

Hover state: slight fade (90% opacity) — no colour shift.

### Logo
- **Icon:** 8-point asterisk / starburst, rounded line ends, no circle background
- **Icon colour:** `#67CC95` (light) or `#6CD79D` (dark)
- **Wordmark:** **AIYA** in Poppins Bold, letter-spacing +0.14em
- **Wordmark colour:** `#171721` on light; `#FFFFFF` on dark

### Layout & shapes
- Corner radius on cards: **16px** (rounded-2xl feel)
- Corner radius on buttons & badges: **full pill**
- Icon badges: circle, 56px, mint or lilac fill at 15% opacity with matching icon colour
- Generous whitespace; max content width ~1152px (6xl)
- Section padding: ~64–80px vertical

### Slide templates to suggest
1. **Dark hero** — `#0A0A0F` bg, white H1, mint CTA pill, purple radial glow top-centre
2. **Light content** — `#E5E2F1` or `#FAFAF9` bg, charcoal text, 3–4 column step cards with mint/lilac icon circles
3. **Mint callout** — gradient mint section, `#082B1B` headings, white/translucent cards with soft shadow
4. **Closing CTA** — lilac tint bg (`#CFBCF4` at 20%), large H2 with Playfair italic accent word, mint button

### Voice & sample copy
- Tagline: *Your AI product team*
- Headline: *Clarity before code.*
- Subhead: *Turn messy ideas into a clear, editable blueprint.*
- Pipeline stages: **Discover → Features → Plan → Blueprint**
- Avoid: “revolutionary”, “disruptive”, corporate stock-photo energy
